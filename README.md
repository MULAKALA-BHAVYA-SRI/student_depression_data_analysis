# Student Depression Data Analysis (Python)

This project performs exploratory data analysis (EDA) on a student depression dataset
to identify patterns and trends using Python.

---

## Objectives
- Clean and preprocess the dataset
- Perform exploratory data analysis
- Visualize mental health related trends
- Derive meaningful, non-clinical insights

---

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---
